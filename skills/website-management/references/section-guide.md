## Section Types by Intent

### Conversion (driving action)

| Section | Best For |
|---------|----------|
| HeroWithMedia | Primary landing page header with side-by-side media |
| HeroWithMediaBg | Immersive full-bleed hero with background image |
| HeroCard | Compact hero with card-style containment |
| HeroWithInputField | Lead capture directly in the hero |
| HeroWithPrompt | Interactive/AI-focused hero with prompt input |
| Banner | Mid-page or end-page call-to-action strip |
| SimpleForm | Standalone contact/lead form |
| FormInCard | Form with card container for visual separation |
| SimpleHero | Clean text-focused hero for minimal designs |

## Page Position Guide

### Landing page (conversion-focused)

1. **Hero** (HeroWithMedia, HeroWithMediaBg, or HeroCard)
2. **Social proof strip** (LogoGrid or MovingLogos) -- establishes credibility early
3. **Features/Value** (StickyFeatures, IconFeatures, or FeaturesWithImage)
4. **Content depth** (CardWithMedia, BentoCards, or MediaBentoCards)
5. **Testimonials** (TestimonialGrid or TestimonialSlider)
6. **Stats** (JustStats or StatsOnCard) -- optional, if metrics strengthen the case
7. **FAQ** (Faqs or FaqsOnCard) -- address objections
8. **CTA** (Banner or SimpleForm)

### Blog/article page

1. **BlogHero** or **HeroWithMedia** (minimal)
2. **ArticleWithToc** or **BlogContent**
3. **Related content** (MediaCards or CardWithMedia)
4. **CTA** (Banner or FormInCard for newsletter)

### About page

1. **SimpleHero** or **HeroWithMediaBg** (team/office photo)
2. **Content blocks** (CardWithMedia alternating for story)
3. **Team** (MediaCards or MediaGridTrio)
4. **Values/Mission** (IconFeatures or TextGrid)
5. **CTA** (Banner)

### Pricing page

1. **SimpleHero** (minimal, value-proposition headline)
2. **PqPricing** or **PqPricingTable**
3. **Faqs** (pricing-specific questions)
4. **TestimonialsWithLogos** (social proof)
5. **Banner** (custom plan CTA)

### Portfolio/case study page

1. **HeroWithMedia** or **HeroWithMediaBg**
2. **MediaBentoCards** or **BentoCards** (work showcase)
3. **StatsOnCard** (project metrics/results)
4. **TestimonialGrid** (client feedback)
5. **Banner** (contact CTA)

## Section Combination Patterns

### "Problem -> Solution" flow
Hero (state the problem) -> Features (your solution) -> Testimonials (proof it works) -> CTA

### "Show, Don't Tell" flow
HeroWithMediaBg (visual impact) -> MediaBentoCards (showcase) -> StatsOnCard (results) -> CTA

### "Trust Ladder" flow
Hero -> LogoGrid (who trusts you) -> Features (why) -> Testimonials (what they say) -> FAQ (address doubts) -> CTA

### "Storytelling" flow
HeroWithMedia -> StickyFeatures (progressive reveal) -> CardWithMedia (deep dive) -> Testimonial (validation) -> Banner
