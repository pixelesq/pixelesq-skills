# Section Visual Guide

All 50+ Pixelesq section types organized by visual effect and use case. Use this to choose sections based on the visual experience you want to create.

## Contents

- Immersive / Full-bleed sections
- Card-based layouts
- Grid layouts
- Scroll-interactive sections
- Carousel / Slider sections
- Minimal / Focused sections
- Navigation sections
- Blog / Documentation sections

## Immersive / Full-bleed

Sections that create strong visual impact, often edge-to-edge.

| Section | Visual Effect |
|---------|--------------|
| HeroWithMediaBg | Full-screen background image with overlay text |
| HeroMediaBgCard | Background image with floating content card |
| HeroWithMediaInset | Hero with inset media container |
| MediaBgCards | Content cards over background media |
| MediaBgIconCards | Icon cards floating over background media |

**Best for:** Landing page openings, portfolio showcases, immersive storytelling. Creates drama and visual weight.

## Card-Based Layouts

Sections that organize content into distinct card containers.

| Section | Visual Effect |
|---------|--------------|
| HeroCard | Hero content in a card container |
| SimpleCard | Single content card |
| CardWithMedia | Content card with side image |
| CardWithMediaInset | Card with inset image container |
| CardsWithIcons | Grid of icon-led cards |
| MediaCards | Uniform media-focused cards |
| MediaGridIconCards | Media grid with icon overlays |
| FaqsOnCard | FAQ accordion in a card |
| StatsOnCard | Metrics displayed in cards |
| FormInCard | Form inside a card container |

**Best for:** Organized, professional layouts. Cards create visual separation and make content scannable. Good for feature lists, services, team members.

## Grid Layouts

Sections using multi-column grid arrangements.

| Section | Visual Effect |
|---------|--------------|
| BentoCards | Asymmetric magazine-style grid |
| MediaBentoCards | Bento grid with heavy media |
| MediaGridAlt | Alternating-size media grid |
| MediaGridTrio | Three-column media grid |
| SimpleMediaGrid | Basic uniform media grid |
| FaqsInGrid | FAQ items in grid layout |
| TextGrid | Text-only content grid |
| IconFeatures | Features in an icon grid |
| TestimonialGrid | Testimonials in masonry grid |
| LogoGrid | Client logos in a grid |

**Best for:** Showcasing multiple items simultaneously. Grids work for portfolios, feature comparisons, logo walls, testimonials. Bento grids add visual interest through asymmetry.

## Scroll-Interactive

Sections that respond to scroll position for progressive content reveal.

| Section | Visual Effect |
|---------|--------------|
| StickyFeatures | Content sticks while features scroll through |
| StickyFeaturesCard | Sticky scroll with card containers |
| StickyFeaturesCardInset | Sticky scroll with inset cards |
| FeaturesOffset | Alternating left-right feature blocks |
| FeaturesWithImage | Feature list with paired large image |

**Best for:** Feature deep-dives, storytelling, product tours. Sticky sections create high engagement by revealing content progressively. Best placed mid-page after the hero.

## Carousel / Slider

Sections with horizontal navigation between content items.

| Section | Visual Effect |
|---------|--------------|
| FeaturesCarousel | Swipeable feature cards |
| TestimonialSlider | Single testimonial with navigation |
| TestimonialCardSlider | Testimonial cards in a slider |
| MediaCardSlider | Media cards with horizontal scroll |
| MediaSliderExtended | Full-width media slider |
| MovingLogos | Auto-scrolling logo strip |

**Best for:** When you have many items but limited vertical space. Carousels work well on mobile. Use for testimonials, feature highlights, partner logos. Moving logos add subtle dynamism.

## Minimal / Focused

Sections with a single clear message and maximum whitespace.

| Section | Visual Effect |
|---------|--------------|
| SimpleHero | Text-focused hero, no distractions |
| HeroWithMedia | Clean hero with side media |
| HeroWithMediaBelow | Hero text above, media below |
| HeroWithMediaItems | Hero with media and item grid |
| HeroWithInputField | Hero with embedded input field |
| HeroWithPrompt | Hero with prompt/search input |
| Banner | Single-line CTA strip |
| JustStats | Standalone metrics display |
| SimpleMedia | Single media element |
| SimpleForm | Clean standalone form |
| Testimonial | Single featured testimonial |

**Best for:** Clarity and focus. Use simple sections when the message is strong enough to stand alone. Great for CTAs, hero areas, and section breaks.

## Pricing

Dedicated pricing section types.

| Section | Visual Effect |
|---------|--------------|
| PqPricing | Pricing cards (2-4 tiers) |
| PqPricingTable | Feature comparison table |

**Best for:** Pricing pages. Cards for simple tier comparison, table for detailed feature-by-feature comparison.

## FAQ

Multiple FAQ layout options.

| Section | Visual Effect |
|---------|--------------|
| Faqs | Standard accordion |
| FaqsOnCard | Accordion inside a card |
| FaqsInGrid | FAQ items in a grid |
| FaqsWithMedia | FAQ with side illustration |

**Best for:** Addressing objections, reducing support load. Place near the bottom of landing pages, above the final CTA. Grid layout works when questions are short.

## Navigation

Header and footer sections.

| Section | Visual Effect |
|---------|--------------|
| HeaderWithMenu | Full navigation with dropdown menus |
| SimpleHeader | Minimal logo + links header |
| FooterWithMenu | Footer with organized link groups |
| SimpleFooter | Basic footer with essentials |
| MegaMenu | Expanded dropdown navigation |

**Best for:** These are typically managed as partials. HeaderWithMenu for sites with many pages, SimpleHeader for minimal sites.

## Blog / Documentation

Content-focused reading layouts.

| Section | Visual Effect |
|---------|--------------|
| BlogHero | Blog listing page hero |
| BlogContent | Blog post body content |
| ArticleWithToc | Long article with table of contents sidebar |
| DocWithSidebar | Documentation with side navigation |

**Best for:** Long-form content. ArticleWithToc is ideal for guides and tutorials. DocWithSidebar for documentation-style content with section navigation.
