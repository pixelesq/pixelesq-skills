# Pixelesq

This plugin connects Claude to Pixelesq for website management. Install the MCP server from `.mcp.json` to enable all tools.

## Skills

Three skills are available that activate based on context:
- **Website Management** — Comprehensive guide for pages, sections, collections, and site structure
- **SEO Optimization** — Search Console analysis, meta optimization, and redirect management
- **Content Creation** — Blog posts, landing pages, and collection entries with brand voice matching

## Agent

Use the **Site Manager** agent (`/pixelesq:site-manager`) for autonomous multi-step website tasks like building complete pages or running site-wide SEO audits.
